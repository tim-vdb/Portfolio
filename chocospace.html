ChocoSpace

HTML

CSS

JS

Site web horizontal, Prototype

Contexte :

Dans le cadre d'un projet académique, l'objectif était de créer un concept marketing et commercial autour d'un bonbon ou chocolat, tout en réalisant également le produit. Avec mon groupe, nous avons imaginé des chocolats en forme de planètes, un concept original et visuellement attractif.

Mission :

Mes contributions principales à ce projet ont été :

  • Développement d'un site web interactif, avec un défilement horizontal unique, sans librairie JavaScript (une seule page en version desktop).
  • Modélisation 3D des chocolats en forme de planètes, pour enrichir l'expérience utilisateur et illustrer notre concept.

Résultat :

Le site propose une navigation innovante et immersive, adaptée au thème galactique de notre produit. La version mobile dispose d'un défilement vertical pour plus de praticité.

Vous pouvez découvrir le site web réalisé pour ce projet via le lien suivant : ChocoSpace

Conseil de navigation

Pour une expérience optimale sur le site web, étant un prototype de site web à défilement horizontal, sans librairie JavaScript. Placez-vous donc bien au centre de la visualisation avant de cliquer sur "Interagir", car le défilement est horizontal en version desktop. Sur mobile, le défilement est vertical pour une meilleure adaptabilité.
